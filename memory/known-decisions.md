# Known Decisions

## Decisiones confirmadas

- El proyecto se basa en Laravel 13 y PHP 8.3.
- El repositorio no contiene un proyecto Next.js completo.
- La lógica de negocio actual usa controladores por dominio y múltiples bases de datos.
- En este punto no existe una arquitectura IA implementada.
- Se requiere documentar y organizar con una estructura viva.

## Decisiones no destructivas

- No se debe refactorizar destructivamente ni eliminar la compatibilidad actual.
- Cualquier mejora debe respetar las rutas y responsabilidades existentes.
- El sistema IA se construirá con agentes documentales y de gobernanza.
